# MIDI file dumper

Simple command-line tool that displays the contents of a MIDI file.

## How to build

1. Open the Command Prompt for VS (Visual Studio).

2. Run the Program Maintenance Utility (NMAKE):

    `nmake`

## How to use

3. Dump a MIDI file:

    `mididump.exe my_midi_file.mid`
